# Customer Churn Analysis
## SQL | Power BI

## Project Overview
An end-to-end customer churn analysis for a telecom company
using SQL and Power BI to identify why customers leave and
the financial impact of churn.

## Problem Statement
A telecom company is experiencing high customer churn
without understanding who is leaving, why, and the revenue
impact. This project provides those answers.

## Tools Used
- PostgreSQL / pgAdmin — ETL & Data Cleaning
- Power BI Desktop — Dashboard & Visualisation

## Project Steps
1. ETL — Imported raw CSV into PostgreSQL
2. Data Cleaning — Handled NULLs, fixed data types
3. SQL Analysis — Churn rate, retention rate, revenue impact
4. Power BI — KPI dashboard with charts and slicers

## Key Findings
- Churn Rate: 26.99%
- Retention Rate: 73.01%
- Month-to-Month contracts have highest churn
- Fiber Optic users churn more than others
- Competitor is the top churn reason
